# src_libs/ - Control Libraries

Reusable libraries for motor

## Main structure
- MCLib/ - Motor control (FOC, observers, etc.)
- BoardLib/ - Board utilities (temperature, protections)
- MainsLib/ - Mains management (PLL, peak detection)

## What to put here:
- Proven and validated control algorithms
- Reusable code between projects
- STM32 optimized implementations
