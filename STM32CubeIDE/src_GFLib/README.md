# src_GFLib/ - General function library

mathematical libraries and general utilities

##structure
- Math/ - Optimized mathematical functions

- Trigonometric functions (CosSinTable)
- Digital filters (LowPass, HighPass, Notch)
- Optimized mathematical functions (sqrt, abs, limit)
- fixed-point arithmetic (frac16, frac32)
- Vector operations and transforms
- Common data types
