# APP/Inverter/ - Inverter Control

Specific control for power inverter

## Mains files:
- InverterControl.c - Main inverter control
- InverterOperation.c - Operation modes
- InverterParameters.h - Specific parameters

## What to put here:
- Inverter state machine (Stop/Start/Run/Error)
- DC - AC power control
- PWM modulation algorihms
- Protection and limits
- Operation modes (manual, automatic, test)
- Specific calibration parameters
